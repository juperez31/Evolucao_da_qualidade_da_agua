# Evolucao_da_qualidade_da_agua
Este projeto analisa a evolução da qualidade da água nos sistemas hídricos de Santos entre 2016 e 2024, com base em dados de monitoramento ambiental. São avaliados parâmetros como oxigênio dissolvido, turbidez, pH, nitrogênio e fósforo, buscando identificar tendências de melhoria ou degradação ao longo do tempo.
